# kottans-frontend
Перше знайомство з гіт-ом подарувало позитивні враження. Напочатку гіт сдається перевантаженним та заплутаним, однак чим більше з ним працюєш, тим легшим він здається (звісно, якщо опрацювати вивченний матеріал на практиці, чи це тільки поки що хах) та одразу розумієш переваги технології VCS і які переваги вона надає, особливо з командними проектами. Стандартний інтерфейс виглядає дещо незручним в представленні для новачка (можна викликати все дерево командною, але, все ж таки, не так зручно), було б чудово, якщо інтерфейс мав графічний вигляд як у грі на learngitbranching.js.org або щось на кшталт цього
